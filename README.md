# dapp-2fa-phone-nav

Web3 hosted dapp with 2FA via phone number. A clean home page with navigation options to play games, NFT auction house,  a gps locator page that connects via google maps, and a social page to chat with friends.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/dapp-2fa-phone-nav.git
cd dapp-2fa-phone-nav
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
